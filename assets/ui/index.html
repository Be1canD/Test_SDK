BLOCK OPS: Reloaded is a fast-paced game mode where every elimination upgrades your weapon, progressing through a set arsenal until the final challenge – a kill with a MEGA baguette – secures victory.

— INSTRUCTIONS —

Use the following commands in the chat box to get going:

/list — shows list of joinable games

/join — join a game by its number (e.g. /join 1)

— CONTROLS —

R KEY for reload
LEFT CLICK to fire
RIGHT CLICK for sniper scope
SHIFT to sprint
SPACE to jump
WASD to move
